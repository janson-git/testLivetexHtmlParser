Пара слов о процессе установки:

0. для корректной работы потребуется PHP 5, c установленными расширениями PDO и PDO_mysql.

1. после распаковки в каталоге с тестовым заданием можно найти файл database.sql. Это изначальный дамп БД, с несколькими, уже выполненными задачами парсинга данных. Для примера.
Необходимо создать БД из этого файла.

2. в корневом файле index.php нужно прописать данные для коннекта к БД, а также - указать значение SERVER_NAME.
Это значение используется, чтобы задание работало, при  установке в подкаталог сервера (например, для "http://localhost/test-livetex/" значение SERVER_NAME будет "/test-livetex/").
